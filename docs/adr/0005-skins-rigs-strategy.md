# ADR 0005: Skins And Rigs Strategy

## Status

Accepted

## Context

The FPS arm work exposed limits in ad hoc meshes and item offsets.

## Decision

Treat the current articulated rig as an experiment on `dev` and build future skins, palettes, held items and animations around a documented rig system.

## Consequences

The FPS arm chantier stays paused until the project foundation is stable. Future work should add visual tests and documented attachment points.
